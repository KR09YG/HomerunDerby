# Home Run Derby

## 概要

個人制作していた野球ゲームからホームランダービー部分を切り出し、
ポートフォリオ用に不要な機能・アセットを整理したプロジェクトです。

ボールの物理計算を自前で実装していることでよりリアルな軌道を実現していることがプッシュポイントです

## 使用技術
- Unity 6 (6000.0.32f1)
- C#
- UniTask
  - https://github.com/Cysharp/UniTask
- DOTween
  - https://dotween.demigiant.com/
- AudioManager
  - https://github.com/kankikuchi/AudioManager
